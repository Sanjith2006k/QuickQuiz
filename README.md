# QuickQuiz

QuickQuiz is a modern, responsive quiz application built with HTML, CSS, and JavaScript. It offers a fast-paced quiz experience with a clean interface, dark/light mode, animated transitions, and detailed performance statistics.

## Features

- Multiple quiz categories
- Easy, Medium, and Hard difficulty levels
- Dark and Light mode
- Countdown timer
- Score and streak tracking
- Progress bar
- Responsive design
- Quiz summary with performance statistics
- Answer review after completion
- Smooth animations and modern UI

## Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla)

## How to Run

1. Clone the repository.
2. Open `index.html` in your browser.
3. Select a category and difficulty to start the quiz.
